Teammates HTML Style Guide

Document Properties
  • Doctype to be used: HTML5
    <!DOCTYPE html>

  • Do not include xml namespace in the document.
    xmlns="http://www.w3.org/1999/xhtml”

  • General structure for HTML document
    <!DOCTYPE html>
    <html>
    <head>
       <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
       <title>Teammates</title>

       <link rel="shortcut icon" href="/favicon.png">
       more stylesheets...

       <script type="text/javascript" src="/js/jquery-1.6.2.min.js"></script>
       more scripts...

    </head>
    <body>
       <div id="dhtmltooltip"></div>
       <div id="frameTop">
         stuff...
       </div>
       <div id=”frameBody”>
         stuff...
      </div>
       <div id=”frameBottom”>
         stuff...
       </div>
    </body>
    </html>

Tags
  • All HTML tags are to be closed properly
    <div>Stuff</div>
    <p>Stuff
    <p>More Stuff

  • Void elements do not have a closing tag. Do not use /> for void elements
    Void elements are: area, base, br, col, command, embed, hr, img, input, keygen, link, meta, param, source, track, wbr
    <br>
    <br />

    <img src=”a” width=”5px” height=”10px”>
    <img src=”a” width=”5px” height=”10px” />

  • Use lowercase for all attributes and tag names
    <div id=”frame”>Test</div>
    <DIV ID=”frame”>Test</div>

Attributes
  • Use [attr=”value”] for attribute values
    <input value=”computer” id=”someInput” disabled=”disabled” tabindex=”2”>
    <input value=computer id='someInput' disabled='disabled' tabindex=2> (single/no quotes)
    <input value="computer" id="someInput" disabled tabindex="2"> (specify boolean variables in the attributes)

  • Avoid using the style attribute within tags. Put them in css sheets instead. If really required, do it in the format style=”attr1: value; attr2: value; attr3: value;”
    <span style=”color: red; display: block; font-size: 15px;”></span>
    <span style=”color:red display:block font-size:15px”> (no whitespace, no semicolon)

Protocols
  • Use relative links rather than absolute links
    <a href=”//www.google.com/”>Google</a>
    <a href=”http://www.google.com/”>Google</a>

Miscellaneous
  • Take note of script tags.
    <script type="text/javascript" src="/js/jquery-1.6.2.min.js"></script>
    <script language="JavaScript" src="/js/jquery-1.6.2.min.js"></script>

  • Do not have trailing whitespaces between opening and closing tags
    <p>Student Name</p>
    <p> Student Name </p>
